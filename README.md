📊 Telecom Customer Churn Analysis Dashboard (Excel Project)
📌 Project Overview

This project presents an end-to-end Telecom Customer Churn Analysis Dashboard built using Microsoft Excel.
The dashboard provides deep insights into customer demographics, subscription behavior, revenue distribution, and churn patterns to help telecom companies improve retention and profitability.

🎯 Business Problem

Customer churn is one of the biggest challenges in the telecom industry.

The objective of this project is to:

Identify key factors driving customer churn

Analyze customer revenue contribution

Understand service adoption behavior

Segment customers based on risk level

Provide data-driven retention insights

🔍 Key Insights

Customers with Month-to-Month contracts have the highest churn rate.

Short-tenure customers are more likely to churn.

Customers without Online Security or Tech Support churn more frequently.

Fiber internet users generate higher revenue.

Electronic check payment method shows higher churn percentage.
